# neatd - User Stories

This directory contains individual user stories organized by epic.

**Total Stories**: 59  
**Epics**: 8 (Epic 0-7)

## Structure

Stories are organized by epic:
- `epic0/` - CLI + Config + Validation (Foundation)
- `epic1/` - Scanner + Entries (Inventory)
- `epic2/` - Policy Engine + Matching
- `epic3/` - Planner (Dry-Run Plan Output)
- `epic4/` - Apply + Audit + Undo
- `epic5/` - Reporting and Status
- `epic6/` - Hardening and Polish
- `epic7/` - Watch Mode (Optional)

## Story Files

Each story is in its own file with format: `EPIC<epic_number>-<story_number>.md`

Example: `EPIC0-1.md` = Epic 0, Story 0.1
